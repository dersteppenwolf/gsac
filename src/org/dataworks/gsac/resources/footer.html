This is GSAC web services for Dataworks, built from the open source code in the main GSAC package on SourceForge. See the installation instructions in the UNAVCO GSAC web site. This code is provided in the standard GSAC package, including the MySQL database Dataworks_GSAC_schema.sql template file.
Update of 24 Feb 2016. SourceForge GSAC revision r1582.

This data repository is powered by UNAVCO's GSAC, a software package for web services. GSAC enables search and download from archives of geoscience data, including GPS/GNSS station information and instrument data files. To learn how to use GSAC and what GSAC can provide, see the UNAVCO GSAC web site, and see the GSAC User Guide. UNAVCO, a non-profit university-governed consortium, facilitates geoscience research and education using geodesy.
