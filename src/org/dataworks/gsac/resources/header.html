Dataworks GSAC Web Services for the GNSS Data Center

To define the GSAC files' header and footer:
Edit the file dataworks/gsac/resources/header.html, for the header of GSAC web site pages.
Edit the file dataworks/gsac/resources/footer.html, for the footer of GSAC web site pages.
Add image files for the header and footer to dataworks/gsac/resources/.
Edit the file dataworks/gsac/htdocs/index.html, for the contents of the GSAC web site home page.
Rebuild GSAC and re-deploy.
