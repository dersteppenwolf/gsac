GSAC Web Services for the GNSS Data Center

To create your GSAC web site HTML files' header and footer:
Edit the file dataworks/gsac/resources/header.html, for the HTML of the header of GSAC web site pages (which originally includes this material). You may include a name and logo image(s) for your agency.
Add image files used by the the header and footer to dataworks/gsac/htdocs/ (not to dataworks/gsac/resources).
Edit the file dataworks/gsac/htdocs/index.html, for the HTML contents of the GSAC web site home page.
Optionally, edit the file dataworks/gsac/resources/footer.html, for the HTML of the footer of GSAC web site pages. The footer provided is good for all GSACs without edits.
Rebuild GSAC and re-deploy.
