GSAC Web Services for the _ Data Center

This is the header for all GSAC web pages.
To create your GSAC web site HTML files:
Edit the file gsac-code/src/org/prototype/gsac/resources/header.html, for the HTML of the header of GSAC web site pages.
Change the two title lines. Include text and images appropriate for your agency.
Edit the file gsac-code/src/org/prototype/gsac/htdocs/index.html, for the HTML contents of the GSAC web site home page.
Optionally, edit the file gsac-code/src/org/prototype/gsac/resources/footer.html, for the HTML of the footer of GSAC web site pages. The footer provided is good for all GSACs without edits. Add image files used by the the header and footer in the directory gsac-code/src/org/prototype/gsac/htdocs/
Rebuild GSAC and re-deploy.
