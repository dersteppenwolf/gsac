GSAC Overview

The GSAC repository Web Services interface allows you to search and browse geodesy and geophysics instrument sites, and to download data files from the instruments. GSAC currently is used for GNSS, DORIS, SLR, and VLBI.

A GSAC website provides complete information about GSAC web services for that repository, with the GSAC client and the GSAC API.
