GSAC Web Services allow you to search and browse a repository for geodesy and geophysics instrument sites, and to download data files from the instruments. GSAC-WS currently is used for GNSS, DORIS, SLR, and VLBI instrument sites and data.

This Help shows how to install GSAC-WS for your data repository, and how to use the GSAC repository web page user interface. Because there are different GSAC repositories, the searches for sites and files differ a little between repositories. The examples show typical cases.

A GSAC repository also provides programmatic access to its repository, using the GSAC API and the GSAC client. See the sections Command Line Client and Using the GSAC Web Service API here. A GSAC respository's capabilities, used by the API, client, and other GSAC repositories, are online; see the Information page for lists of the REST arguments for the APi and client.
