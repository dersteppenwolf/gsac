GSAC File Downloader

See also the File search results section, the previous Help page.

The GSAC file downloader is a Java Webstart based application that is used to download to your computer the results of a GSAC files query. It lists the FTP URLS and will download each file and place it in the specified Destination Directory.

Make local directories

Checking the Make local directories checkbox results in the downloader making subdirectories under the Destination Directory that correspond to the directory paths in the FTP URLs.

Overwrite files

Normally, if a file already exists on your computer then the downloader will not fetch a new one. You can control this behavior with the Overwrite files option.
