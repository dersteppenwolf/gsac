UNAVCO European-Caribbean Federated GSAC

This Federated GSAC uses UNAVCO's GSAC software for federated data search and downloads, merging GSAC web services from six GSACs in Europe, the TLALOCNet GSAC in Mexico, and the CIMH COCONet GSAC in Barbados. UNAVCO cannot guarantee that all these are operational at all times.

For how to use GSAC to find and download geoscience data, see the User Guide for GSAC Data Repositories.

To begin searches for site information, click on Search Sites above, or click on Search Files to find instrument data files available to download.
